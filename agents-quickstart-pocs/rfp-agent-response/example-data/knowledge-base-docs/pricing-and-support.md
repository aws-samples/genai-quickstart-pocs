# Pricing and Support Models

## Implementation Pricing
### Core Banking Platform
- **Tier 1 Banks** (>$10B assets): $2.5M - $5M implementation
- **Tier 2 Banks** ($1B - $10B assets): $1M - $2.5M implementation  
- **Community Banks** (<$1B assets): $500K - $1M implementation
- **Credit Unions**: $250K - $750K implementation

### Payment Processing Platform
- **Setup Fee**: Waived for qualified merchants
- **Integration Fee**: $25K - $100K depending on complexity
- **Certification Fee**: $15K for card network certifications
- **Custom Development**: $200/hour for specialized requirements

## Ongoing Costs
### Licensing and Maintenance
- **Core Banking**: 18-22% of license fee annually
- **Payment Processing**: 2.65% + $0.10 per transaction
- **Cybersecurity Services**: $15K - $50K monthly retainer
- **Cloud Infrastructure**: AWS costs + 15% management fee

### Support Tiers
- **Basic Support**: Business hours, email/phone support
- **Premium Support**: 24/7 support with 4-hour response SLA
- **Enterprise Support**: Dedicated support team with 1-hour response SLA
- **White Glove**: On-site support and dedicated customer success manager

## Training and Professional Services
### Training Programs
- **Administrator Training**: $2,500 per person (5-day program)
- **End User Training**: $500 per person (2-day program)
- **Train-the-Trainer**: $5,000 per person (3-day program)
- **Custom Training**: $1,500 per day for on-site delivery

### Professional Services
- **Project Management**: $250/hour for certified PMP resources
- **Technical Consulting**: $200/hour for solution architects
- **Data Migration**: $150/hour for data specialists
- **Integration Services**: $175/hour for API developers

## Volume Discounts
- **10% discount** for contracts >$1M annually
- **15% discount** for contracts >$5M annually
- **20% discount** for contracts >$10M annually
- **Custom pricing** for strategic partnerships and multi-year commitments
